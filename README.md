# Brain-Tumor
This project classifies brain MRI images into tumor types using Support Vector Machine with Principal Component Analysis for feature reduction. Different kernels are compared and experiments are conducted with varying training data sizes. Model performance is evaluated using accuracy, F1-score, sensitivity, and specificity.
